# 🌌 Portfolio – Joaquín Ignacio Clark

---

## 🚀 Características principales

- ⚡️ Construido con **React 19** + **Vite 7** + **Tailwind 4**
- 🛰️ Enfoque en **backend, DevOps, IoT/OT** y sistemas críticos
- 🧠 **Filtrado dinámico de proyectos** (backend, infra, IoT)
- 💬 Secciones: *Inicio*, *Sobre mí*, *Proyectos*, *Contacto*

---

## 🧩 Tecnologías

- [React 19](https://react.dev/) + [react-router-dom 7](https://reactrouter.com/)
- [Vite 7](https://vitejs.dev/)
- [Tailwind CSS 4](https://tailwindcss.com/) (vía `@tailwindcss/vite`)
- [react-icons](https://react-icons.github.io/react-icons/), [typewriter-effect](https://www.npmjs.com/package/typewriter-effect)

---

## 📂 Estructura del proyecto

```text
/
├── public/                # favicon, imágenes estáticas
├── src/
│   ├── assets/            # imágenes importadas por componentes
│   ├── components/        # componentes React (commons, home, about, project, links)
│   ├── data/              # datos estáticos (proyectos, jobs)
│   ├── pages/             # rutas (Home, About, Proyectos, Contact, NotFound)
│   ├── styles/            # CSS global y por sección
│   ├── App.jsx            # router y layout
│   └── main.jsx           # entry point
├── index.html
└── vite.config.js
```

## 🧞 Comandos

Todos los comandos se ejecutan desde la raíz del proyecto:

| Comando            | Acción                                               |
| :----------------- | :--------------------------------------------------- |
| `npm install`      | Instala las dependencias                             |
| `npm run dev`      | Levanta el servidor de desarrollo en `localhost:5173` |
| `npm run build`    | Compila la versión de producción a `./dist/`        |
| `npm run preview`  | Previsualiza el build localmente                    |
